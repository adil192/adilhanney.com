# [adilhanney.com](https://adilhanney.com)

This is the source code for my personal website using React + TypeScript + Vite.

We use Deno as the package manager: see [the docs](https://deno.com/) to install Deno.

## Commands

### Build

```bash
deno install
deno run build
```

### Develop

```bash
deno install
deno run dev
```

### Update dependencies

```bash
deno update --latest
```
